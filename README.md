# 🎬 React Movie App (Vite + React)

A modern and responsive movie browsing application built using **React + Vite**.  
This app allows users to explore movies, view details, and manage favorites with a smooth UI and fast performance ⚡

Important for Indian users:
The movie API (e.g., MDba) is sometimes blocked in India.

---

## 🚀 Features

- 🔍 Search for movies in real-time  
- 🎞️ View movie posters, titles, and details  
- ❤️ Add / remove movies from favorites  
- 💾 Persistent favorites using localStorage  
- ⚡ Fast development with Vite (HMR enabled)  
- 🎨 Clean and responsive UI  

---

## 🛠️ Tech Stack

- **Frontend:** React (Hooks + Context API)
- **Build Tool:** Vite
- **State Management:** Context API
- **Styling:** CSS
- **API:** Movie Database API
